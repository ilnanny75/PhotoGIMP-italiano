# Come contribuire a PhotoGIMP-italiano 🇮🇹

Grazie per l'interesse nel migliorare questa versione italiana di PhotoGIMP! Il contributo di tutti è benvenuto per rendere l'esperienza di GIMP sempre più vicina a quella professionale.

## Come puoi aiutare?
1. **Segnalazione Bug:** Se lo script `installa_italiano.sh` non trova la tua cartella GIMP, apri una "Issue" segnalando il tuo sistema operativo e il percorso della cartella.
2. **Miglioramento Traduzioni:** Se trovi errori nei file della cartella `docs` o nei messaggi dello script, sentiti libero di correggerli.
3. **Test su diverse versioni:** Aiutaci a testare la patch su GIMP 2.10, 3.0 e versioni Flatpak/Snap.

## Processo di Pull Request
* Fai un fork del progetto.
* Crea un branch per la tua modifica (`git checkout -b modifica-fantastica`).
* Fai il commit delle tue modifiche (`git commit -m 'Aggiunta nuova funzione'`).
* Invia il branch (`git push origin modifica-fantastica`).
* Apri una Pull Request verso questo repository.
