# PhotoGIMP (Versione Italiana) 🇮🇹

Questa è una patch personalizzata per GIMP (2.10 o 3.0+) che trasforma l'interfaccia, le scorciatoie e i plugin per renderli simili ad Adobe Photoshop. Ideale per chi è abituato all'ecosistema Adobe e vuole passare all'open-source senza traumi.

---

## ✨ Novità di questa versione
* **Traduzione Completa:** Documentazione e istruzioni interamente in italiano.
* **Script Intelligente:** Il file `installa_italiano.sh` rileva automaticamente la tua versione di GIMP (sia essa Flatpak o installazione standard) e applica la patch nel percorso corretto.
* **Editor Predefinito:** Lo script imposta automaticamente **Geany** come editor di testo predefinito per il sistema e per la sincronizzazione.

---

## 🚀 Installazione Rapida
Per installare la patch sul tuo computer, apri il terminale all'interno di questa cartella e lancia i seguenti comandi:

```bash
chmod +x installa_italiano.sh
./installa_italiano.sh
